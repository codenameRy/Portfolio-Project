TV Digital

Project #2

Description: TV Digital is a single page web application built featuring thousands of TV shows, allowing users to login and save their favorite shows to an account dashboard. The user accounts are managed on the server-side with user and show models to link the favorite shows with the unique user in the dashboard. CRUD (Create, Read, Update, and Delete) application using the MERN Stack.

Technologies: JavaScript, React.js, Node.js, Express.js, MongoDB, Mongoose, Ant Design, HTML, CSS, GitHub, Heroku
